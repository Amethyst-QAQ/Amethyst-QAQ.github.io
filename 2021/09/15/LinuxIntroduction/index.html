Linux操作系统了解

2021-09-15

Linux简介

Linux是一种自由和开放源码的操作系统，存在着很多不同的Linux版本。Linux可安装在各种计算机硬件设备中，比如手机、平板电脑、路由器、台式计算机等

Linux的特点：多用户，多任务，丰富的网络功能，可靠的系统安全，良好的可移植性，具有标准兼容性，良好的用户界面，出色的速度性能，且是开源的

Linux命令

目录结构

  • bin（binaries）存放二进制可执行文件
  • sbin（super user binaries）存放二进制可执行文件，只有root才能访问
  • etc（etcetera）存放系统配置文件
  • usr （unix shared resources）用于存放共享的系统资源
  • home 存放用户文件的根目录
  • root 超级用户目录
  • dev （devices）用于存放设备文件
  • lib （library）存放跟文件系统中的程序运行所需要的共享库及内核模块
  • mnt （mount）系统管理员安装临时文件系统的安装点
  • boot 存放用于系统引导时使用的各种文件
  • tmp（temporary）用于存放各种临时文件
  • var（variable）用于存放运行是需要改变数据的文件

Linux命令

命令分类

命令格式：command -options parameter

例如：

ls -la /usr

内部命令：属于Shell解析器的一部分

cd            切换目录（change directory）
pwd            显示当前工作目录（print working directory）
help        帮助

外部命令：独立于Shell解析器之外的文件程序

ls            显示文件和目录列表（list）
mkdir        创建目录（make directory）
cp            复制文件或目录（copy）

查看帮助文档

内部命令：-help + 命令        例：-help cd
外部命令：-man + 命令        例：-man ls

常用命令（概述）

ls                查看当前文件夹下的内容           list
pwd                查看当前所在文件夹            print work directory
cd [目录名]      切换文件夹                     change directory
touch [文件名]      如果文件不存在，新建文件        touch
mkdir[目录名]      创建目录                      make directory
m[文件名]          删除指定文件                remove
clear            清屏                         clear

tips：

  • ctrl + shift + = 放大终端窗口的字体显示
  • ctrl + - 缩小终端窗口的字体显示
  • 输入文件/目录/命令的前几个字母之后，可以使用tab键自动补全：没有歧义会直接自动补全；有歧义时系统会提示可能存在的命令
  • 上/下光标键可以在之前使用过的命令之间来回切换
  • ctrl + c 用于退出选择并停止执行当前命令

Linux命令及使用方法

查找文件

find命令功能非常强大，通过用来在特定目录下搜索符合条件的文件（如果省略路径，表示在当前文件夹下查找）

find [路径] -name “*.py”            查找指定路径下拓展名是.py的文件，包括子目录

软链接

（注意源文件要使用绝对路径，不能使用相对路径）

ln -s 被链接的源文件 链接文件            建立文件的软链接（类似Windows下的快捷方式）

硬链接（了解即可）

ln 被链接的源文件 链接文件                建立硬链接文件，两个文件占用相同大小的硬盘空间

P.S. 在Linux中，文件名和文件的数据是分开储存的

打包压缩

Linux常用的打包压缩方式是tar.gz

# 打包文件
tar -cvf 打包文件.tar 被打包的文件/路径
# 解包文件
tar -xvf 打包文件.tar

tar选项说明：

c            生成档案文件，创建打包文件
x            解开档案文件
v            列出归档解档的详细过程，显示进度
f            指定档案文件名称，f后面一定是.tar文件，所以必须放选项最后

软件安装

通过apt安装/卸载软件

  • apt是指Advanced Packaging Tool，是Linux下的一款安装包管理工具
  • 可以在终端中方便的安装/卸载/更新软件包
# 安装软件
sudo apt install 软件包
# 卸载软件
sudo apt remove 软件包
# 更新已安装的包
sudo apt upgrade

查看目录内容：ls

  • ls（英文list的缩写），功能为列出目录的内容
  • Linux文件/目录 名称最长可以有256个字符参数
  • 以 . 开头的文件为隐藏文件，需要用 -a 参数才能显示
  • … 代表上一级目录
  • 常用参数：
-a          显示指定目录下所有子目录与文件，包括隐藏文件
-l          以列表的方式显示文件的详细信息
-h          配合-l以人性化的方式显示文件的大小
  • ls通配符的使用
*           代表任意个数个字符
?           代表任意一个字符
[]          表示可以匹配字符组中的任意一个
[abc]       匹配a/b/c中的任意一个
[a-f]       匹配从a到f范围内的任意一个字符

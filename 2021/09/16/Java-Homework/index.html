小禾のJava程序设计大作业

2021-09-16

小组成员

  • 组长：Amethyst（田心禾）
  • 组员：清蒸大白虾（甘蔚芳）、奶油菠萝包（王玥）、

设计及成果要求

  • 设计要求：

    • 用java开发一个区块链平台，提供基本的区块链服务
    • 在该平台上能实现NFT的功能
    • NFT能对接现实生活中的物品

  • 成果要求：

    • 提交能运行的软件及源代码
    • 需要开发文档，包括需求文档、项目设计方案、发展进程报告及使用手册等

设计思路

  • 该项目为策略经营类游戏，旨在以魔法世界为背景，运用区块链技术，打造一个虚拟空间。
  • 主要玩法为，玩家通过炼金术（和计算机高级语言java相结合）攀爬科技树，在此过程中可以获得道具、虚拟货币和经验，提升等级。
  • 玩家使用虚拟货币可以从其他玩家处购买所需要的部分道具，或者解锁更高级的炼金术。玩家也可以将闲置的虚拟物品按照指定价格挂出，其他玩家支付虚拟货币并购买。
  • 玩家也可以通过接口定价挂出经过认证的现实物品，其他玩家可以使用虚拟货币购买。

设计目标

本游戏旨在通过游戏的方式让玩家学习java语言的相关知识，同时达到娱乐自身的精神需求，建设精神世界。同时玩家可以通过游戏中的虚拟货币购买现实物品，可以帮助玩家实现自身价值，在娱乐和现实中找到平衡。

需求概述（暂定）

  • 用户需求：

    • 玩家可以通过本游戏获取java语言的相关知识，同时达到娱乐自身的精神需求，建设精神世界
    • 玩家可以通过游戏中的虚拟货币购买现实物品，可以帮助玩家实现自身价值，在娱乐和现实中找到平衡
    • to be continue……

  • 功能需求：

    • 等级和经验：玩家通过获得经验提升等级，进而解锁更多游戏功能
    • 炼金术：基于计算机高级语言java设计。玩家通过所给的指引学习并掌握方法，使用基础道具通过炼金术获得高级道具、虚拟货币和经验
    • 虚拟货币：玩家可以通过虚拟货币从其他玩家处购买所需要的部分道具以及部分炼金方法，也可以将暂时闲置的道具出售为虚拟货币；同时玩家也可以通过接口定价挂出经过认证的现实物品，其他玩家可以使用虚拟货币购买
    • 商铺经营：达到一定等级后，玩家会拥有自己的商铺。可以在其中挂出暂时闲置的道具或者通过接口定价挂出经过认证的现实物品，等待其他玩家使用虚拟货币购买

  • 前端将使用Unity游戏引擎制作
  • 后端将使用计算机高级语言java开发，使用区块链技术实现
